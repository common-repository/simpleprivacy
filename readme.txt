A simple way to create a private Wordpress blog with htauth.

== Description ==
This will protect the user or admin facing wordpress pages. Separate login and passwords can be setup for each. 

This isn't anything fancy, not super secure, just basic and easy. It is called simple after all.

== Installation ==

Use the plug-in installer yo!

== Frequently Asked Questions ==

= How secure is this? =

It works, beyond that no promises here. Feel free to let me know if you have ideas for making it more secure.
